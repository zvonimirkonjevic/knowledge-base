# knowledge-base

A personal playground for learning different frameworks and technologies. Each
top-level directory is a self-contained project — small experiments, code
snippets, and worked-through tutorials, each with its own setup and dependencies.

## Projects

| Project | What it covers |
|---------|----------------|
| [sqlalchemy_unified_tutorial/](sqlalchemy_unified_tutorial/) | Working through the [SQLAlchemy 2.0 Unified Tutorial](https://docs.sqlalchemy.org/en/20/tutorial/index.html) — transactions, the DBAPI, Core/ORM data manipulation, and querying against a Dockerized PostgreSQL database. |

## Structure

Every project is isolated: it brings its own README, dependencies, and (where
relevant) Docker setup. Start from a project's own README for instructions on how
to run it.
